## 摘要
实时单目同步定位与地图构建（SLAM）从根本上存在尺度模糊性以及缺乏几何自校正的问题。虽然3D高斯泼溅（3DGS）能够实现高保真渲染，但现有的仅RGB系统仍为开环系统，因为深度先验被注入到建图过程中，但优化后的几何无法有效调节跟踪漂移。我们提出了MyGO-Splat，这是一种闭环高斯SLAM框架，能够将高斯基元解析地光栅化为逐像素深度和表面法线，从而使地图能够主动监督相机位姿优化。为桥接单目先验和尺度一致性，我们的框架引入了尺度感知自适应对齐，将基础模型深度估计投影到全局优化后的高斯空间中，形成尺度反馈的自校正循环。大量评估表明，这种闭环设计提高了尺度稳定性和外观-几何一致性，在使用仅单目输入时实现了与RGB-D方法相当的性能。

## Abstract
Real-time monocular Simultaneous Localization and Mapping (SLAM) fundamentally suffers from scale ambiguity and a lack of geometric self-correction. While 3D Gaussian Splatting (3DGS) enables high-fidelity rendering, existing RGB-only systems remain open-loop because depth priors are injected into mapping but refined geometry cannot effectively regulate tracking drift. We present MyGO-Splat, a closed-loop Gaussian SLAM framework that analytically rasterizes Gaussian primitives into pixel-wise depth and surface normals, allowing the map to actively supervise camera pose optimization. To bridge monocular priors and scale consistency, our framework introduces scale-aware adaptive alignment that projects foundation-model depth estimates into the globally optimized Gaussian space, forming a self-correcting cycle for scale feedback. Extensive evaluations show that this closed-loop design improves scale stability and appearance-geometry consistency, achieving performance comparable to RGB-D methods while using only monocular input.

## 1. 核心问题与研究动机

### 背景问题
- **实时单目SLAM** 存在固有的**尺度模糊性**和**缺乏几何自校正**能力。由于没有度量深度，长期几何一致性十分脆弱。
- 现有仅RGB的高斯SLAM系统多为**开环**：跟踪模块基于短时匹配估计位姿，建图模块利用多视图监督优化几何，但优化后的几何**并不参与调控跟踪过程**，导致尺度漂移无法被修正。
- 3D高斯泼溅（3DGS）虽能提供高质量渲染，但在仅RGB设置下，单目深度先验（如从基础模型获得）存在尺度不一致和帧间偏差，直接注入优化会产生浮动伪影、表面扭曲等问题。

### 本文目标
提出一个**闭环几何反馈框架**，使高斯地图从被动的渲染器升级为能主动监督跟踪的几何组件，在不依赖深度传感器的情况下，实现尺度稳定、外观-几何一致的实时重建。

## 2. 方法论

### 核心思想
- 将高斯SLAM重构为**闭环系统**：通过解析光栅化从高斯原语中导出像素级深度和表面法线，使地图能主动参与相机位姿优化。
- 引入**尺度感知自适应对齐**：将基础模型（VGGT）的单目深度先验投影到全局优化后的高斯空间中，形成自校正的尺度反馈循环。
- 结合**多目标联合优化**：同时约束渲染外观、解析几何、对齐深度先验以及表面法线一致性。

### 关键技术细节

#### 2.1 在线跟踪
- **基于光流的位姿估计**：采用DROID-SLAM的循环神经网络预测光流，构建关键帧图，通过密集局部BA优化位姿和深度。
- **闭环检测**：使用EigenPlaces提取全局几何特征，通过FAISS数据库检索历史帧，检测到回环后执行全局BA，消除累积漂移。

#### 2.2 闭环几何反馈（CLG Feedback）
- **几何先验生成**：使用VGGT（3D视觉基础模型）在单次前向传播中生成深度图。
- **不确定加权度量同步**：对每个关键帧，计算最优尺度λ和偏移τ，将VGGT深度对齐到高斯渲染深度，最小化加权最小二乘误差。对齐后的深度替代原始噪声深度，反馈回全局BA，形成自校正循环。

#### 2.3 基于光栅化高斯几何的建图
- **光栅化像素深度**：受RADE-GS启发，在高斯原语光栅化过程中，利用射线-高斯相交几何，通过平面方程计算每个像素的精确深度，避免中心投影近似。
- **表面解析法线**：从射线空间中的平面方程解析导出法线，并变换回相机坐标系，提供几何监督。
- **几何增强多目标优化**：优化函数包含四个损失项：
  - 外观损失 \(\mathcal{L}_{rgb}\)：渲染图像与输入RGB之间的L1损失。
  - 深度损失 \(\mathcal{L}_{d}\)：渲染深度与对齐后深度之间的L1损失。
  - 深度不连续性正则项 \(\mathcal{L}_{dis}\)：沿同一射线的不同高斯深度差异最小化，减少伪影。
  - 法线一致性损失 \(\mathcal{L}_{n}\)：解析法线与从深度导出的法线之间的余弦相似度损失。
  - 总损失：\(\mathcal{L} = \lambda_{rgb}\mathcal{L}_{rgb} + \lambda_{d}\mathcal{L}_{d} + \lambda_{dis}\mathcal{L}_{dis} + \lambda_{n}\mathcal{L}_{n}\)，超参数经实验调整。
