## 摘要
重复求解参数化偏微分方程（PDE）对于不确定性量化、设计优化和反问题至关重要，但传统的神经算子需要昂贵的非凸训练。我们提出了PCA-RaNN，一种随机潜在神经算子，结合了基于PCA的降维、固定随机特征和闭式最小二乘读出。它将潜在算子学习重新表述为固定特征线性回归，将训练时间减少一到三个数量级，同时保持竞争力精度。我们引入了一种能量匹配缩放规则和一个轻量级的双参数BFGS修正，以纠正次优的特征尺度。集成平均减少了预测方差。在Burgers、Darcy、Navier-Stokes和反向热方程基准测试中，PCA-RaNN相对于算子学习基线提供了良好的速度-精度权衡。该集成支持分裂保形预测区间，线性读出通过递归最小二乘实现快速在线适应，无需重新训练隐藏特征。为多查询科学工作流提供了一种高效的不确定性感知代理。

## Abstract
Repeatedly solving parametric PDEs is essential for uncertainty quantification, design optimization and inverse problems, but conventional neural operators require expensive non-convex training. We introduce PCA--RaNN, a randomized latent neural operator that combines PCA-based dimensionality reduction with fixed random features and a closed-form least-squares readout. It recasts latent operator learning as fixed-feature linear regression, reducing training time by one to three orders of magnitude across benchmarks while maintaining competitive accuracy. We introduce an energy-matched scaling rule and a lightweight two-parameter BFGS refinement to correct suboptimal feature scales. Ensemble averaging reduces predictive variance. On Burgers, Darcy, Navier--Stokes and backward heat equation benchmarks, PCA--RaNN provides a favorable speed--accuracy trade-off against operator-learning baselines. The ensemble supports split-conformal prediction intervals, and the linear readout enables rapid online adaptation via recursive least squares without retraining hidden features. This provides an efficient, uncertainty-aware surrogate for many-query scientific workflows.

- **研究动机**：参数化偏微分方程（PDE）的重复求解是许多科学计算任务（如不确定性量化、设计优化、反问题）的核心，但传统的数值方法（有限元、有限差分等）在大量查询场景下计算成本高昂。
- **背景痛点**：近年兴起的神经算子（如 DeepONet、FNO）虽然能高效学习从参数到解的映射，但其深层网络训练依赖非凸优化，需要大量时间和算力，且对初始化、超参数敏感；同时许多科学工作流还需要不确定性估计和快速模型更新能力。
- **总体目标**：提出一种既能保持预测精度又大幅降低训练成本，并能提供不确定性量化和在线适应能力的轻量级神经算子。

# 2. 方法论

### 核心思想
将神经算子学习“解耦”为三个环节：
1. 利用 PCA 将高维输入/输出场压缩为低维潜在坐标；
2. 用**固定随机特征**（两层隐藏层 + 跳跃连接）对潜在映射进行非线性变换，仅输出层权重通过最小二乘拟合；
3. 解码回物理空间。  
这使得整个潜在算子学习变为**线性回归问题**，避免了反向传播和非凸优化。

### 关键技术细节
- **PCA 编码/解码**：对输入和输出分别用 centered PCA 提取低维坐标（潜在维度 d_A, d_U）。
- **随机潜在回归器**：  
  - 隐藏层权重 \(W^{(1)}, W^{(2)}\) 从均匀分布采样后**固定**；  
  - 缩放参数 \(r_1, r_2\) 通过“能量匹配规则”初始设定（根据潜在数据能量分配比率），避免网格搜索；  
  - 可选的 **BFGS 微调**：仅优化两个对数尺度参数 \(\eta_1, \eta_2\)（\(r_l = \bar{r}_l \exp(\eta_l)\)），最小化验证集损失，输出层仍为最小二乘/岭回归。
- **集成预测**：K 个独立随机特征成员的预测均值降低方差。
- **不确定性量化**：采用**分裂保形预测**（split conformal prediction），基于校准集计算非一致性分数，得到分布自由的边际预测区间；无需假设预测分布形式。
- **在线适应**：利用线性读出的结构，通过**递归最小二乘（RLS）** 增量更新输出权重，无需重新训练 PCA 基或隐藏层。

### 算法流程（文字描述）
1. 用训练数据建立 PCA 编码器/解码器，将每个样本映射为潜在对 \((x_i, y_i)\)。
2. 根据能量匹配规则计算初始缩放 \((\bar{r}_1, \bar{r}_2)\)，可选 BFGS 微调。
3. 随机采样并固定隐藏层权重，构建特征矩阵 \(H\)；通过最小二乘求解输出权重 \(V\)。
4. 集成：重复步骤 3 K 次，各成员分别有独立随机权重，但共享缩放参数（BFGS 版本）。
5. 预测时，对新样本先 PCA 编码，再经随机特征提取，最后用集成均值得出预测。
6. 不确定性量化：用校准集计算保形分位数，构建预测区间。
7. 在线适应：固定隐藏层，用 RLS 更新输出权重。

# 6. 主要结论与发现

1. **训练速度大幅提升**：在四个基准上，PCA-RaNN 训练时间比对比方法减少 1–3 个数量级（如 Burgers 从数小时降至秒级）。
2. **精度竞争力**：在多数问题中误差与全训练神经算子相当或更好；在反向热方程上 BFGS 微调版本超越所有基线。
3. **能量匹配缩放规则有效**：避免了耗时的网格搜索，且 BFGS 微调可进一步纠正次优缩放（尤其在病态问题上改进显著）。
4. **集成提供稳定的不确定性量化**：分裂保形预测区间在经验上接近名义覆盖率（约 95%），区间宽度与问题复杂性相关。
5. **线性读出支持快速在线适应**：在 Navier-Stokes 时域迁移任务中，仅用 200 个新样本通过 RLS 即可将误差降低 72.8%，更新时间仅 0.6 秒（批量重拟合需 4.5 秒）。
6. **总体定位**：PCA-RaNN 不是要替代所有神经网络算子，而是在计算预算有限、需要快速重新部署或不确定性输出时的轻量替代方案。

# 7. 优点

- **理论支撑**：附录提供了单隐藏层 RaNN 高概率逼近定理，说明随机特征可替换确定性网络而不损失渐近表示能力。
- **实践友好**：能量匹配 + 两参数 BFGS 微调避免了繁琐的手动调参；闭式解使训练极其快速且稳定。
- **多功能性**：同一框架同时支持快速训练、不确定性量化（保形预测）、在线适应（RLS）。
- **模块化设计**：PCA、随机特征、缩放选择、集成、共形预测均可独立替换或增强，易于扩展。
- **实验结果全面**：覆盖正问题与反问题，对比多个主流基线，并提供了组件分析。

# 8. 不足与局限

- **网格依赖性**：PCA 编解码器基于训练离散化，无法直接实现网格无关评估或连续插值，限制了在变几何问题上的应用（与扩散映射算子等方法相比）。
- **物理信息融合困难**：当前框架难以直接融入 PDE 残差约束（因为没有物理空间的连续表示）。
- **非线性映射能力有限**：在高度非线性或长时程湍流动力学（如 Navier-Stokes）中，精度低于全训练神经算子（FNO）。
- **理论分析不完整**：逼近定理仅针对单隐藏层 RaNN，未覆盖文中使用的两层 + 跳跃连接架构，也未分析 PCA 缩并和 BFGS 缩放的影响。
- **实验范围有限**：仅在四个 2D/1D 基准上验证，未涉及 3D 问题、弱间断问题、大规模多物理场场景；在线适应实验仅测试了 Navier-Stokes 的一种时间迁移情形。
- **BFGS 微调的成本**：虽然只有两个参数，但需在验证集上迭代计算梯度，对于能量匹配已足够好的问题（如 Darcy）改进不大，却增加了少许时间。
- **集成成员共享缩放**：BFGS 微调在一次随机实现上完成缩放后，所有集成成员使用相同缩放，可能导致非最优的成员个体性能。
