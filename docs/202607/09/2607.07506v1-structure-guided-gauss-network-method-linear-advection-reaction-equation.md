## 摘要
文献[5]中针对线性平流-反应方程引入的最小二乘神经网络（LSNN）方法能够无需先验界面位置知识而精确逼近不连续解。然而，由此产生的离散化是一个非凸优化问题，计算强度大且复杂。本文提出一种结构引导高斯-牛顿（SgGN）方法，该方法在线性（输出）参数和非线性（隐藏层）参数之间交替迭代。在每个外循环中，线性参数通过线性求解器计算，非线性参数通过改进的高斯-牛顿（GN）方法更新，该方法显式移除GN矩阵的奇异性。文献[5]中所有测试问题的数值实验表明，SgGN方法不仅在计算成本上，更重要的是在精度上均优于常用的Adam优化器[13]。

## Abstract
The least-squares neural network (LSNN) method introduced in [5] for linear advection-reaction equations is capable of accurately approximating discontinuous solutions without a priori knowledge of the interface location. However, the resulting discretization is a non-convex optimization problem that is computationally intensive and complex. In this paper, we propose a structure-guided Gauss-Newton (SgGN) method that alternates between the linear (output) and the nonlinear (hidden layer) parameters. At each outer iteration, the linear parameters are computed by a linear solver, and the nonlinear parameters are updated by a modified Gauss-Newton (GN) method that explicitly removes the singularities of the GN matrix. Numerical experiments for all test problems presented in [5] show that the SgGN method is superior to the Adam optimizer [13], the commonly used first-order optimization algorithm, not only in computational cost but, more importantly, in accuracy

- **研究动机**：克服 LSNN 方法中非凸优化计算复杂、精度低的困难，设计一种兼顾效率与精度的优化算法。

## 2. 方法论：结构引导高斯-牛顿方法（SgGN）
### 核心思想
- 利用浅层 ReLU 网络参数的可分离性，将优化问题分解为**线性参数（输出层权重 c）** 与**非线性参数（隐藏层权重与偏置 r）** 的交替更新（块非线性高斯-赛德尔外循环）。

### 关键技术细节
1. **线性参数更新**：每次外迭代中，固定 r，损失函数关于 c 是二次型，通过求解对称正定线性系统直接得到全局最优：
   \[
   c^{(k)} = A(r^{(k)})^{-1} F(r^{(k)})
   \]
   其中 \(A(r)\) 是刚度矩阵，\(F(r)\) 是右端项。

2. **非线性参数更新**：固定 c，采用改进的 Gauss-Newton 方法：
   - 推导损失函数关于 r 的梯度 \(\nabla_r L_\tau\) 和 GN 矩阵 \(G(c,r)=(D(\hat{c})\otimes I_3) H(r) (D(\hat{c})\otimes I_3)\)。
   - 识别 GN 矩阵奇异性仅来源于对应线性参数 \(c_i=0\) 的神经元。定义活跃神经元集合 \(I = \{i: |c_i| \ge \epsilon_c\}\)，仅对活跃神经元构建**约化 GN 方向**：
     \[
     d_R = (D(\hat{c})\otimes I_3)_I^{-1} H_I(r)^{-1} (D(\hat{c})\otimes I_3)_I^{-1} (\nabla_r L_\tau)_I
     \]
     非活跃神经元方向置零。

3. **算法流程**（Algorithm 4.1）：
   - 初始化 r\(^{(0)}\)（如均匀划分断线）。
   - 对每个外迭代 k：
     - 求解线性参数 c\(^{(k)}\)；
     - 计算约化 GN 方向 p\(^{(k)}\)；
     - 更新 r\(^{(k+1)} = r^{(k)} + p^{(k)}\)。

4. **理论基础**：建立了梯度与积分关于 Heaviside 函数的交换性质（Lemma 3.2），保证了 GN 矩阵推导的合法性。证明了刚度矩阵 A 和层 GN 矩阵 H 的正定性（Theorem 4.6）。

## 6. 主要结论与发现
- **SgGN 方法在精度和效率上全面优于 Adam**：
  - 垂直界面问题：50 次迭代相对误差 8.29×10⁻¹³（Adam 20,000 次迭代误差 5.80×10⁻²）。

- **几何解释**：SgGN 能高效旋转和平移断线至正确界面位置，即使初始化不完美（如平行于β）也能快速调整。
- **网络结构影响**：增加神经元数量并不一定能减少迭代次数或提高精度（如对角问题中 n 更大时停止标准更难满足），但固定迭代次数下误差随 n 增大而下降。

## 7. 优点
- **方法创新性**：
  - 利用 ReLU 网络参数分离结构，设计交替优化，线性参数直接求解，避免非凸问题。
  - 首次显式移除 GN 矩阵因零线性参数导致的奇异性，保持搜索方向的正确性（不同于盲目正则化的 LM）。
- **理论基础扎实**：建立了启发/狄拉克函数与梯度的交换性质，严格证明矩阵正定性。
- **实验结果突出**：在多个不连续解问题上达到接近机器精度的准确性，且无振荡或过冲。
- **对初始化鲁棒**：即使初始断线远离最终界面（如平行于 β），SgGN 也能有效旋转和移动。
